
# Python Quiz Game

## Project Description
This is a simple Python-based quiz game.  
The game asks 5 multiple-choice questions and scores your correct answers.  
It is easy to run and modify for your own quiz needs.

---

## How to Run

1. Make sure you have Python installed (version 3.x recommended).  
2. Download the file `quiz_game.py`.  
3. Open your terminal or command prompt.  
4. Navigate to the folder containing `quiz_game.py`.  
5. Run the command:
   ```
   python quiz_game.py
   ```
6. Follow the on-screen instructions to answer the questions.

---

## Requirements

- Python 3.x (Download from https://www.python.org/downloads/)
- No external libraries needed.

---

## Features

- 5 multiple-choice questions.  
- Immediate feedback on your answer (Correct or Wrong).  
- Final score display at the end.
